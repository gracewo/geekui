# geekui
 
轻量级pc端 Vue3 组件库
    

## 特性

* 70+ 高质量组件（3.0 持续开发中）
* 支持按需引用
* 详尽的文档和示例
* 支持 TypeScript
* 支持定制主题


## 支持环境

* vite2 + ts + vue3


## 安装使用

具体安装使用方法请参考 GeekUI 官网 

## ⌨️ 本地开发

```bash
$ git clone https://github.com/gracewo/geekui.git
$ cd geekui
$ npm install
$ npm run dev
```
